# OpenWRT 24.10 NCS601W Config

This repository contains modifications to the OpenWRT Wansview NCS601W configuration to get it compatible with 24.10. It's being developed and tested on two Belkin Netcam F7D7601v1 model cameras but should work with the Wansview NCS601W, imogenstudio +CAM, and Aztech WIPC303 as they appear to be clones.
